A simple skeleton for PHP katas.

## Requirements
- One of the following:
  - PHP 8.3+
  - Docker
- Make (optional)

## Usage with Docker

Build the image.

```shell
make build
```

Run the tests.

```shell
make test
```

## Usage with local PHP

Install dependencies.

```shell
composer install
```

Check everything works.

```shell
php vendor/bin/phpunit
```
